1. Introduction

Interactive Patient Care System is an Online Web Application used by patients, doctors and Hospital Staff.

1.1 Purpose

The purpose of this software is to provide an option for patients to fill forms online which saves time for patients. It will allow patients to fill forms online, and help doctors to have all records of the patients in a database and be able to view them on a computer or mobile device at any time.  

1.2 Software Environment

  • Windows 2000 & above with JDK 1.5.
  • Apache Tomcat  server.
  • Internet browser like Mozilla ,Internet Explorer,Google Chrome.

1.3 User Access Levels

All the users of this of this application will have different access levels to all the functionality of the application. Certain types of users will only have access to some functionalities.

1.3.1 Patient

Patient can login in, fill forms, and submit the forms to doctor. Patient do not have access to register other patients to the website.

1.3.2 Doctor or Hospital Staff

Hospital Staff or the Doctor can register patients to the website. They can view the forms filled by the patients. Doctor or Hospital Staff also has access to fill patient forms for special patients.

1.3.3 Others

Can only see the homepage of the website where contact information of the hospital is available and perform any operations other than just knowing contact information

1.4 About this volume

 

This volume is written to help explain the use of Interactive Patient Care System.  It contains the procedures that must be followed to fulfill the application.

2. Patient Module

2.1 Fill Patient Forms and Submit them

Patients log in to the application and fill the forms under treatment request and submits the forms. The submitted forms should reach the appropriate hospital staff and doctors.

2.1.1 Navigate to Login Page

  1. Launch the browser on computer or mobile
  2. Navigate to address bar of the browser
  3. Type the url of the website in the browser and press Enter button
  4. Home page of the website will be displayed
  5. Click on the Login button available at the top right corner of the website
  6. Navigation to Login page takes place

2.1.2 Login to the application as a patient

  1. Click in the Username text field and then enter the Username in the text field.
  2. Click in the Password text field and then enter the password in the field
  3. Navigation to Home page of the patient takes place

2.1.3 Navigation to Fill ESAS Record

  1. Home page of patient contains  Treatment Request link
  2. Click on the Treatment Request link
  3. Will be navigated to page where two hyperlinks are available namely Record and Create Appointment
  4. Click on Record Hyperlink

2.1.4 Fill ESAS Record and Submit It

  1. Only accessible after completing the 2.1.2 or 2.1.4 Section. On the webpage, the ESAS Record and Record Body Part buttons are available . Click on ESAS Record button.
  2. Now fill up  record by choosing the “Pain”, “Tiredness”, “Anxiety”, etc and the scale of each symptom.
  3. Enter all the mandatory fields and optional fields can also be filled its upto to the user.
  4. Now click on submit button
  5. A confirmation message will be displayed or an error message
  6. If an error message is displayed it means form is not submitted have to do modifications according to message and click submit again
  7. If OK is pressed in confirmation page the Patient will be again navigated to Record Page where

2.1.5 Fill Record Body Part form and Submit It

  1. After 2.1.2 or 2.1.4 Section. Now on the webpage the Patient can see ESAS Record and Record Body Part buttons are available . Click on Record Body Part button.The user will be navigated to Record Body Part Page where a diagram of human body is available. Now point out the pain in the diagram and give the rating of pain.
  2. Click on submit button at the end of  the page
  3. A confirmation message will be displayed or an error message
  4. If an error message is displayed it means form is not submitted have to do modifications according to message and click submit again
  5. If a confirmation message is received it means form submitted successfully.

3. Doctor or Hospital Staff Module

3.1 Register patient to website

Doctor or Hospital staff logins to the application and registers the patient to the application and gets username and password for the patient

3.1.1 Navigate to Login Page

Refer to Section 2.1.1

3.1.2 Login to application as Doctor or Hospital Staff

  1. Click in the Username text field and then enter the Username in the text field.
  2. Click in the Password text field and then enter the password in the field
  3. Navigation to Home page of the doctor takes place

3.1.3 Fill Patient Details

  1. On Home page of Doctor Register Patient link is available
  2. Click on the Register Patient link
  3. Will be navigated to a web page to fill all the details of the patient and filed to create username and password
  4. Fill the fields and submit the page.

3.1.4 Check the confirmation message

  1. Submit the page after filling all the details of the patient
  2. Error message will be displayed if all mandatory fields are not submitted
  3. Confirmation message will be displayed if the registration is successful

4. Working with Interactive Patient Care System

4.1 Navigating across Fields

  • Can navigate using move
  • Can navigate using arrows on keyboard
  • Can navigate using shortcuts
  • Navigate using buttons on visible on web page. Do Not use arrows of browsers to navigation may lose the user’s sessions

4.2 Using Shortcut Keys

  • End button -- End of page
  • Tab Key -- Move to next field

4.3  ToolBars and ToolTips

  • Application contains standard header and footer which acts a toolbar and by clicking header can be navigated to home page
